Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM :
1. Elements by Tag Name
2. Elements by Class Name
3. Elements by ID Name
4. By QuerySelector
5. By QuerySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: The innerHTML gives me the inner HTML with texts also of the selected element.
innerText: The innerText gives me the visible text of the selected element.
textContent: The textContent gives me all the text of the selected element as well as the display none texts.

Question-3: What is event delegation in the DOM?

When we add a event listener in the parent node it also applies the event listener to the child nodes also by using even bubbling. It also checks which child node was clicked. (If my event listener is click event.)

Question-4: What is event bubbling in the DOM?

When we click a element using event listener the event bubbles to the parent element, then again bubbles to the parent element across the site. One after another.

Question-5: How do you create, add, and remove elements using JavaScript?

For creating a element we can use document.createElement, for adding the created element we have to use appendChild() to add the element in the site and to remove the element we have to use remove() function for the selected element.
